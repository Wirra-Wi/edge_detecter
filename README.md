# 邊緣偵測應用程式

這是一個使用 Streamlit 建立的本地端圖像邊緣偵測應用程式，支援多種邊緣偵測演算法和參數調整。

## 功能特色

- 📤 **圖片上傳**：支援 JPG, JPEG, PNG 格式
- 🔍 **三種邊緣偵測方法**：
  - Canny 算子
  - LoG (高斯拉普拉斯) 算子
  - Sobel 算子
- 🎛️ **三種參數模式**：
  - 自然影像：針對一般照片優化的預設參數
  - 醫學影像：針對醫學圖像 (X-ray, MRI 等) 優化的預設參數
  - 自訂參數：手動調整各演算法參數
- 🧹 **影像去噪**：支援非局部均值去噪 (NLM)
- 📊 **即時參數調整**：側邊欄提供所有參數控制
- 💾 **結果下載**：處理後圖片可直接下載

## 參數說明

### 自然影像模式
針對一般照片優化的預設參數組合。

### 醫學影像模式
針對醫學圖像優化的預設參數，包含較低的閾值和適中的去噪。

### 自訂參數模式
手動調整各演算法參數：

#### Canny 算子
- **高斯模糊 kernel 大小**：預處理模糊程度 (1-15, 步長2)
- **minVal**：邊緣下限閾值 (0-255)
- **maxVal**：邊緣上限閾值 (0-255)

#### LoG (高斯拉普拉斯) 算子
- **高斯模糊 kernel 大小**：預處理模糊程度 (1-15, 步長2)
- **閾值**：二值化閾值 (0-100)

#### Sobel 算子
- **高斯模糊 kernel 大小**：預處理模糊程度 (1-15, 步長2)
- **Sobel kernel size**：Sobel 核大小 (1, 3, 5, 7)
- **閾值**：二值化閾值 (0-255)

### 影像去噪
- **啟用去噪**：開關控制
- **去噪強度 h**：非局部均值去噪強度 (1-20)

## 使用方法

1. **啟動應用程式**
   ```bash
   streamlit run app.py
   ```
   在瀏覽器中開啟 http://localhost:8501

2. **上傳圖片**
   - 點擊檔案上傳器選擇圖片
   - 支援 JPG, JPEG, PNG 格式

3. **選擇參數模式**
   - 自然影像：適用一般照片
   - 醫學影像：適用醫學圖像，預設啟用去噪
   - 自訂參數：手動調整所有參數

4. **選擇邊緣偵測方法**
   - Canny：適合精確邊緣偵測
   - LoG：適合斑點和邊緣偵測
   - Sobel：適合梯度強度邊緣偵測

5. **調整參數** (自訂模式)
   - 在側邊欄調整相應參數
   - 參數即時生效

6. **啟用去噪** (可選)
   - 在側邊欄勾選「啟用去噪」
   - 調整去噪強度

7. **執行處理**
   - 點擊「執行」按鈕
   - 查看處理結果

8. **下載結果**
   - 點擊「下載處理後圖片」按鈕
   - 檔案格式為 PNG

## 技術細節

### 邊緣偵測流程
1. **預處理**：灰階轉換 + 去噪 (可選)
2. **模糊**：高斯模糊降噪
3. **邊緣偵測**：應用選擇的演算法
4. **後處理**：閾值二值化
5. **輸出**：二值邊緣圖

### 效能考量
- 大圖片處理時間較長
- 去噪功能會增加處理時間
- 建議圖片大小控制在 1920x1080 以內

## 故障排除

### 應用程式無法啟動
- 確認 Python 版本 ≥ 3.7
- 確認所有依賴已安裝
- 檢查端口 8501 是否被占用

### 圖片處理失敗
- 確認圖片格式支援 (JPG/PNG)
- 檢查圖片檔案是否損壞
- 嘗試調整參數或關閉去噪

### 記憶體不足
- 減少圖片解析度
- 關閉去噪功能
- 重新啟動應用程式
